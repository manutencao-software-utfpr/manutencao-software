	\subsection{1) Situação do problema ou do novo requisito}
	\begin{itemize}
		\item mod1 Anúncios estão ocupando grande parte da tela (Corretiva)
		
		Situação: Este problema dificulta a visualização dos cardapios.
		\item mod2 Adicionar campo para buscar por cidade na tela inicial (Perfectiva)
		
		Situação: Esta melhoria facilita a busca por cidades.
		\item mod3 Remover label estabelecimentos fechados e abertos (Corretiva)
		
		Situação: Como não está funcionando como deveria, será removido.
		\item mod4 Remover carrinho de compras (Corretiva)
		
		Situação: Esta correção é necessária pois não estamos permitindo realização de pedidos ainda.
		\item mod5 Melhorar o layout do footer (Perfectiva)
		
		Situação: Para que o usuário possa encontrar os links ativosdo site com facilidade é necessária essa melhoria.
		\item mod6 Mostrar tempo estimado de entrega (Perfectiva)
		
		Situação: Para facilitar a escolha de qual estabelecimento será escolhido pelo cliente.
		\item mod7 melhorar o layout dos cardápios, como produtos são mostrados no cardápio (Perfectiva)
		
		Situação: Os produtos não possuem imagens, logo a imagem do produto será removida.
		\item mod8 Links quebrados (Corretiva)
		
		Situação: Os links do footer não estão funcionando corretamente.
		\item mod9 Migrar o front end para Vue js. (Perfectiva)
		
		Situação: Para melhorar a estrutura do código e facilitar futuras modificações.
		\item mod10 Migrar o back end para Node js. (Perfectiva)
		
		Situação: Para melhorar a estrutura  do código e facilitar futuras modificações.
	\end{itemize}
